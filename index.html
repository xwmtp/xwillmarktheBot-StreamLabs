Commands for xwillmarktheplace’s chat

Most commands will only work when my bot is online, which is usually when I’m streaming.

Records

Records will be fetched from Speedrun.com. When no argument is given, the record of the current category on stream will be returned. The bot looks at the stream title to figure out which category it is. Make sure to use Speedrun.com user names.

These commands currently work for all Ocarina of Time categories (both main and extensions).

Command Result Example use
!pb Get streamer’s pb for the current category on stream, or for a specific category. !pb, !pb no im/ww
!userpb Get pb of a specific user, for the current stream or a specific category. !userpb juwk, !userpb jelster no wrong warp
!wr Get the world record for the current category on stream, or for a specific category. !wr, !wr nocturne rta
!bingopb Get the bingo pb of the streamer or a specific user. !bingopb, !bingopb scaramanga

SRL races and bingos

These commands will get information from SpeedrunsLive.com about the race that the streamer is competing in, and bingo statistics. The bot is able to work with SRL, SRC, Twitch and Twitter names.

Command Result Example use
!card Get the link to the bingo card in the current race. !card
!race Get the list of entrants in the current race, and a link to the SpeedrunsLive page. !race
!average get the average of the last x bingos of the streamer or a user. Uses x=15 by default. !average, !average kintefleush, !average 5 psymarth
!median Like average, but gets the median instead. Takes the average of the middle values for odd numbers. !median, !median kintefleush, !median 10 psymarth
!results Get the results of the last x bingos. !results, !results kintefleush, !results psymarth 7

Quotes

These are handled by Nightbot, which means they always work.

Command Result Example use
!quote Get a random quote. Add a number for a specific quote, or add a search term to look for a quote. !quote, !quote 3, !quote bang
!addquote Add a quote. Can only by used by regulars in the chat. !addquote “Zora tunic sucks” ~ xwmtp, 2019
!clip Get a random clip. Add a number for a specific clip, or add a search term to look for one. !clip, !clip 5, !clip hess
!jelstertip Get a random Jelster tip. !jelstertip
