Chat commands

Commands and usage for xwillmarkheBot, a Twitch bot for Ocarina of Time speedrunners (available for download here).

Records

Records will be fetched from Speedrun.com. When no argument is given, the record of the current category on stream will be returned. The bot looks at the stream title to figure out which category it is.

These commands currently work for all Ocarina of Time categories (both main and extensions).

Command Result Example use
!pb Get streamer’s pb for the current category on stream, or for a specific category. !pb, !pb no im/ww
!userpb Get pb of a specific user, for the current stream or a specific category. !userpb juwk, !userpb jelster no wrong warp
!wr Get the world record for the current category on stream, or for a specific category. !wr, !wr nocturne rta

SRL races

These commands will get information from SpeedrunsLive about the race that the streamer is competing in.

Command Result
!race Get the list of entrants in the current race, and a link to the SpeedrunsLive page.
!card Get the link to the bingo card in the current race.
!goal Get the link to race goal.
!entrants Get all the entrants in the race.

SRL results

Get stats on past SpeedrunsLive races of the streamer or other users. The bot is able to work with SRL, SRC, Twitch and Twitter user names.

Command Result Example use
!average Get the average of the last x races of the streamer or a user. Uses x=15 by default. !average, !average kintefleush, !average 5 psymarth
!median Like average, but gets the median instead. Takes the average of the middle values for odd amounts. !median, !median kintefleush, !median 10 psymarth
!results Get the results of the last x races. !results, !results kintefleush, !results psymarth 7
!pb Add a race type as an argument to get the streamer’s best SRL result. !pb bingo, !pb short-bingo, !pb blackout, !pb rando, !pb srl
!userpb Same as !pb, but for a specific user. !userpb bonooruu rando
